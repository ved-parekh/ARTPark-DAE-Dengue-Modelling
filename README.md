# ARTPark-DAE-Dengue-Modelling
Modelling Dengue using XGBoost for ARTPark Technical Assignment. This repository contains all the code for data preprocessing and feature engineering in the notebooks: 'Rainfall Data' and 'Urbanization'. The last notebook 'xgboost' contains the model. All datasets used and created are attached here.
